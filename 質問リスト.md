### RESEND_API_KEY の設定
本番環境でお問い合わせメールを実際に送信するには Resend API キーが必要です。

設定先:
- Vercel: `vercel env add RESEND_API_KEY production`
- ローカル: `.env.local` に `RESEND_API_KEY=re_...`

Resend アカウント: https://resend.com

（API キーは Resend ダッシュボードで発行する必要があり、こちらから設定できません）
